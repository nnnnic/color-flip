Color Flip
==========

##Converts #HEX to RGBa or vice versa.

Will insert the converted value wherever your cursor is, or replace selected text.

####HEX to RGBa:
  1. Initiate Alfred
  2. Enter the HEX value leading with #

####RGB(a) to #HEX
  1. Initiate Alfred
  2. Enter the RGB(a) value leading with rgb...
